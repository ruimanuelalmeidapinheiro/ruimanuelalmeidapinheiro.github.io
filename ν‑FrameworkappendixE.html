Appendix E: Cross‑Check Tables

Version-Locked 2024-06-01 Data

E.1 Purpose

Provide side‑by‑side comparisons between ν‑framework frequencies and independent, non-overlapping measurements extracted from different laboratories, techniques, or physical principles.

Each table states the cross‑check observable, the ν‑framework prediction (derived via f = E / h), the external measurement, and the discrepancy Δ.

All discrepancies are within 3σ combined uncertainty unless explicitly flagged; any > 3σ entry would constitute a framework falsification.

E.2 Legend

Z, A — Nuclide identifier
Component — ν‑vector channel being tested
Predicted (Hz) — Value from authoritative energy via exact CODATA 2022 h
External (Hz) — Independent experiment not using primary database
Δ (ppm) — Discrepancy in parts‑per‑million
σcomb — Combined 1σ uncertainty (ppm)
✓ ≤ 3σ, ✗ > 3σ
